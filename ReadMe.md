## Canifern

Caninfern is the culmination of a project I started during my P.h.D at IST back in Portugal. At some point, after a ton of meetings 
discussing how to write a control loop that guarantees a specific timing, I went back into hibernation and came up with the Cainfern
architecture. The software itself is fairly simple, you have three distinct processes, namely: a sensor reader; a watchdog; a controller.
